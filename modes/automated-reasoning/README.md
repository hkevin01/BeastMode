# Automated Reasoning Mode

## Overview

The Automated Reasoning mode, also known as "Beast Mode 3.1," is a powerful autonomous agent designed to solve complex problems through extensive research, codebase investigation, and iterative development.

## Files

- **`primary-solver.agent.md`** - Modern VS Code agent format (recommended for current use)
- **`profile.md`** - Configuration profile with detailed workflow instructions
- **`legacy-format.chatmode.md`** - Legacy format maintained for backward compatibility

## Usage

Activate this mode when you need:
- Multi-step problem solving with autonomous iteration
- Extensive internet research with caching awareness
- Codebase exploration and refactoring
- Implementation testing and validation
- Edge case identification and handling

## Key Workflow Steps

1. **Research** - Fetch URLs and gather all available information
2. **Understand** - Break down problem into manageable parts
3. **Investigate** - Explore codebase and identify root causes
4. **Plan** - Develop detailed step-by-step implementation plan
5. **Implement** - Make incremental, testable code changes
6. **Test** - Rigorously verify all edge cases and scenarios
7. **Validate** - Ensure comprehensive solution completeness
