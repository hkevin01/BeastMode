# Architecture & Microservices Design Mode

## Overview

The Architecture & Microservices Design mode specializes in system architecture, high-level design, microservices decomposition, and scalability planning.

## Files

- **`architect.agent.md`** - Architecture & microservices designer

## Target Audience

- Solutions architects
- System architects
- Technical leads
- Infrastructure/DevOps architects
- Microservices specialists
- CTO/VP Engineering

## Architecture Dimensions

- **Current Assessment**: Component boundaries, dependencies, bottlenecks
- **Requirements Analysis**: Functional and non-functional requirements
- **Architectural Patterns**: Monolith, Microservices, Event-driven, CQRS
- **Microservices Decomposition**: Service boundaries, data ownership
- **Scalability Design**: Horizontal/vertical scaling strategies
- **Reliability**: High availability, disaster recovery, fault isolation

## Design Deliverables

1. **Architecture Assessment Report**: Current state, strengths, weaknesses
2. **Target Architecture Design**: Components, data flow, services
3. **Microservices Decomposition Plan**: If applicable
4. **Implementation Roadmap**: Phased migration approach
5. **Operational Considerations**: Deployment, monitoring, management
6. **Technology Stack Recommendations**: Frameworks, databases, tools

## Scalability Analysis

Provides:
- Current bottleneck identification
- 12-month and 5-year load projections
- Scaling strategies for each constraint
- ROI comparison for different approaches
- Technology recommendations for scale

## Reliability & Resilience

Includes:
- Failure mode analysis
- Detection and recovery strategies
- Multi-region deployment options
- Circuit breakers and graceful degradation
- High-availability design patterns

## Use Cases

- System architecture review
- Microservices decomposition planning
- Scalability planning (10x growth)
- Technology stack selection
- Multi-region or multi-cloud strategy
- Disaster recovery planning
- Performance optimization at architectural level
- High-availability design
